# Organization App

Organization portal for managing campaigns, tasks, and submissions.

## Features
- Campaign management (create, view)
- Task management (view, manage submissions)
- Submission approval/rejection
- Member management
- Organization settings and profile

## Environment Variables
- `NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY`
- `CLERK_SECRET_KEY`
- `DATABASE_URL`

## Development
```bash
pnpm dev
```

## Build
```bash
pnpm build
```
